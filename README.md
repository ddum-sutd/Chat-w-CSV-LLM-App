# Chat with CSV using Gemini Pro

This repository contains a Streamlit application that allows users to chat with a CSV file using the Gemini Pro model.

## Requirements

- Python 3.8 or higher
- Streamlit
- Langchain
- Chroma
- Google Generative AI
- Pandas
